# Gemini Multimodal Studio 🧬

A high-performance creative suite powered by **Google Gemini 3** and **Veo**. This application is a professional-grade tool for creators, offering neural voice cloning, cinematic video generation, 4K image forging, and deep media understanding.

## 🚀 Key Features

### 🎙️ Advanced Voice Synthesis (TTS)
- **Neural Identity Mapping**: Clone any voice from a short audio clip.
- **Stable TTS Engine**: Optimized to prevent server errors (500) using natural language prompting.
- **Expressive Controls**: Fine-tune pitch, cadence, and prosody (Whisper, Dramatic, etc.).

### 🎬 Veo Video Generation
- **Cinematic Production**: Generate 1080p-ready videos from simple text prompts.
- **Aspect Ratio Mastery**: Supports both 16:9 (Landscape) and 9:16 (Portrait/TikTok) outputs.

### 🧠 Media Intelligence (Urdu & English)
- **Deep Extraction**: Provide any video URL or file to get a full content breakdown.
- **Verbatim Transcripts**: Automatically detects and extracts **Urdu and English** transcripts.
- **RTL Support**: Native Right-to-Left (RTL) rendering for Urdu text.

### 🎨 Image Forge
- **4K Masterpieces**: Generate images using the Gemini 3 Pro engine.
- **Grounding Support**: High-fidelity details grounded in real-world knowledge.

## 🛠 Tech Stack
- **Frontend**: React 19, Tailwind CSS, TypeScript.
- **AI Integration**: Google GenAI SDK.
- **Aesthetics**: Glassmorphism UI with high-performance animations.

## 📦 Getting Started

1. **API Key**: Ensure you have a valid Gemini API key from a paid project for Veo and Pro features.
2. **Environment**: The app expects `process.env.API_KEY` to be available.
3. **Run**: Simply serve the root directory using a local server (`npx serve .`).
